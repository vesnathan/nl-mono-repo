## Tips for best results:

1. **Bing Image Creator (FREE):**
   - Go to https://www.bing.com/images/create
   - Sign in with Microsoft account
   - Paste prompt, click "Create"
   - Generate 4 variations, pick best
   - Download as PNG

2. **Leonardo.ai (FREE tier):**
   - Sign up at https://leonardo.ai/
   - Use "Leonardo Diffusion XL" model
   - Style preset: "Illustration" or "Cartoon"
   - Set to "Portrait" orientation
   - Free tier: 150 tokens/day

3. **After generating:**
   - Remove background using remove.bg (free)
   - Save as PNG with transparency
   - Place in `/public/assets/avatars/` folder
   - Name files: `drunk-danny.png`, `clumsy-claire.png`, etc.

4. **If characters don't match exactly:**
   - Try regenerating 2-3 times
   - Adjust prompt slightly (add "more exaggerated" or "simpler style")
   - Pick the one that best captures the personality
