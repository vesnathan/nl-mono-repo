# Audio Generation Prompts for Dealer Characters

This document contains prompts for generating text-to-speech audio files for each dealer character using AI voice generation tools like ElevenLabs, Play.ht, or similar services.

## General Guidelines

- **Format**: MP3 or WAV, 44.1kHz sample rate
- **Duration**: Keep callouts brief (1-3 seconds)
- **Tone**: Professional but personable, casino-appropriate
- **Volume**: Normalize to -3dB to prevent clipping
- **Background**: Clean audio, no background noise

## Technical Specifications for AI Generation

### Recommended Settings:

**ElevenLabs:**
- Stability: 50-60% (allows for natural variation)
- Clarity + Similarity Enhancement: 75-80%
- Style Exaggeration: 0-20% (depending on character)
- Speaker Boost: On

**Play.ht:**
- Voice Speed: 1.0x (normal) to 1.1x (slightly faster for Sharp Eye)
- Voice Pitch: 0 (neutral) to +2 (slightly higher for Rookie)
- Voice Emotion: Match character profile

**Murf.ai:**
- Emphasis: Medium
- Pitch: Adjust per character
- Speed: 100-110 WPM

### Post-Processing:

1. **Normalize** audio to -3dB
2. **Trim** silence at start/end (leave 0.1s padding)
3. **Compress** slightly to even out dynamics
4. **High-pass filter** at 80Hz to remove rumble
5. **Export** as MP3 320kbps or WAV 44.1kHz/16-bit

## Background Sounds & Environmental Audio

### Casino Ambience (Looping Backgrounds)

These are continuous background tracks that create the casino atmosphere. Use AI audio generation tools like Mubert, Soundraw, or AIVA for ambient soundscapes.

#### 1. Casino General Ambience (Looping, 2-3 minutes)

**Prompt for AI Audio Generator**:
```
Create a realistic casino floor ambient soundscape with the following elements:
- General crowd murmur and conversations (30% prominence)
- Distant slot machine sounds in background (20% prominence)
- Occasional player exclamations like "Yes!", "Come on!", laughter (15% occurrence)
- Soft footsteps on carpeted floor (10% prominence)
- Subtle background music (10% prominence) - jazz or lounge style
- Ice clinking in glasses occasionally (5% occurrence)
- Chair movements and general movement sounds (10% prominence)

Mood: Professional casino atmosphere, medium energy, not overwhelming
Tempo: Slow to medium ambient
Style: Realistic, immersive
Duration: 2-3 minute loop
Volume balance: Background layer, shouldn't overpower foreground sounds
```

**Alternative Manual Approach**:
- Record or find casino ambience from royalty-free sources
- Layer multiple tracks: crowd murmur, distant slots, footsteps
- Mix at 15-25% volume relative to dealer voice

## Audio Implementation Notes

### Mixing Levels
- Dealer voice: 30-40% (clear, primary)
- Casino ambience: 15-25% (background)
- Slot machines: 15-20% (background)
- Event sounds: 40-90% (depending on event)
- Character voices: 35-45% (dialogue, important)

### Layering Strategy
1. **Base layer**: General casino ambience (continuous)
2. **Mid layer**: Slot machines (continuous)
3. **Top layer**: Event sounds (triggered)
4. **Front layer**: Dealer/character voices (primary focus)

### Performance Considerations
- Compress audio files to MP3 320kbps or less
- Use Web Audio API for layering
- Preload essential sounds, lazy load event sounds
- Implement audio pooling for repeated sounds
